# 딴타임 (ddantime)

자기 이해와 관리에 초점을 둔 딴짓 기록 앱 백엔드

## 프로젝트 개요

- Spring Boot + JPA 기반 REST API 서버
- 도메인 중심 구조
- Swagger(OpenAPI) 적용

## 기술 스택

- Java 17
- Spring Boot 3.4.5
- Spring Data JPA
- PostgreSQL
- SpringDoc OpenAPI (Swagger UI)

## 로컬 실행 방법

```bash
# .env 파일 작성
DB_URL=jdbc:postgresql://localhost:5432/ddantime
DB_USERNAME=...
DB_PASSWORD=...
```
